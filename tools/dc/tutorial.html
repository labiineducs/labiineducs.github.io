¿Que es Dibujar en la Cuadrícula?

¡Pintar cuadraditos en una grilla con javascript!

La función pintar() nos permite colorear el cuadrado donde se encuentra el cursor.

El cursor esta representado por ✏️.

Ejecuta el programa con 🗘 para visualizar la acción que realiza pintar(). Puedes reiniciar la cuadrícula a su estado inicial con ⭮

Podemos mover el cursor...

Las funciones izquierda(), derecha(), abajo() y arriba() nos permite mover el cursor al cuadradito vecino (en la dirección indicada).

¡Probalo!

Puedes usar secuencias de estas instrucciones para hacer dibujos en la cuadrícula.
